# BullNook 金牛图标设计

> 设计日期：2026-07-05
> 状态：待实现

## 目标

为 BullNook（牛角尖）iOS App 设计并生成一张全新的 1024×1024 应用图标 PNG，以“金牛”为主视觉形象，替换现有的抽象牛角图标。

## 设计决策

### 整体构图

- 画布尺寸：1024 × 1024 px（正方形，Xcode 会自动处理圆角遮罩）。
- 主体位于画面中心偏上，下方放置品牌文字，整体重心稳定。

### 风格

- **3D 金属雕塑感**：使用渐变、高光和阴影模拟金属光泽，而非真实 3D 渲染。
- 视觉关键词：贵气、稳重、专业、吉祥。

### 金牛姿态

- **侧脸回头**：牛头朝左、微微回首，牛角向后弯曲，轮廓饱满。
- 寓意：沉稳霸气，像传统金牛雕塑，符合金融类 App 的稳重感。

### 色彩

- **背景**：金色径向渐变，中心亮金 `#FFD700`，向外过渡到古铜金 `#B8860B` → `#8B6914`。
- **金牛**：深古铜金 `#8B4513` / `#A0522D` 为主，额头、鼻梁、牛角脊背加亮金 `#FFD700` 高光。
- **文字**：浅金色/近白色 `#FFF8DC`，带轻微投影确保可读性。

### 品牌文字

- 图标底部居中放置 “BullNook” 英文品牌名。
- 使用现代无衬线大写字体风格（代码绘制时通过 Pillow 默认无衬线字体实现）。

## 技术方案

- **实现方式**：Python Pillow 代码绘制。
- **步骤**：
  1. 创建 1024×1024 画布。
  2. 绘制金色径向渐变背景。
  3. 使用贝塞尔曲线/多边形绘制侧脸回头金牛轮廓。
  4. 为牛角、额头、鼻梁添加线性渐变高光，模拟金属质感。
  5. 添加底部阴影增强立体感。
  6. 绘制 “BullNook” 文字。
  7. 导出为 `app_icon_1024.png`（覆盖原文件），同时保留 Python 源文件便于后续迭代。

## 输出文件

- `app_icon_1024.png`：最终 1024×1024 应用图标。
- `scripts/generate_app_icon.py`：图标生成脚本，便于后续调整颜色、姿态、文字。

## 验收标准

- [ ] PNG 尺寸为 1024×1024，无透明像素或四角已做圆角预览处理。
- [ ] 图标中清晰可见金牛侧脸形象。
- [ ] 文字 “BullNook” 在金色背景上清晰可读。
- [ ] 整体色调以金色/古铜金为主，无突兀颜色。
- [ ] 生成脚本可重复运行并产出一致结果。

## 后续可扩展

- 若 3D 效果不够满意，可基于本设计稿交给专业设计师或 AI 绘图工具做照片级渲染。
- 未来可扩展生成完整 Xcode App Icon Set（含 20pt/29pt/40pt/60pt 等尺寸）。
