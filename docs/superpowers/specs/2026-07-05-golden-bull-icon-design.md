# BullNook 金牛图标设计

> 设计日期：2026-07-05
> 状态：已实现

## 目标

为 BullNook（牛角尖）iOS App 设计并生成一张全新的 1024×1024 应用图标 PNG，以“金牛”为主视觉形象，替换现有的抽象牛角图标。

## 设计决策

### 整体构图

- 画布尺寸：1024 × 1024 px（正方形，Xcode 会自动处理圆角遮罩）。
- 主体位于画面中心，正面牛头占据主要视觉区域，下方放置品牌文字。

### 风格

- **3D 金属雕塑感**：使用渐变、高光、镜面反光和阴影模拟金属光泽，而非真实 3D 渲染。
- **抽象几何剪影**：正面牛头 + 牛角，线条简洁、符号感强，适合小尺寸图标识别。
- 视觉关键词：贵气、力量、金属、吉祥。

### 金牛姿态

- **正面牛头**：对称构图，牛头居中，强调视觉稳定感。
- **牛角朝上**：牛角从额头两侧向上翻起，尖端朝上，呼应“牛角尖”品牌名，整体呈力量感。
- 表情：略带愤怒感的眉骨、瞪眼、大鼻孔与鼻环，增强公牛的辨识度。

### 色彩

- **背景**：金色径向渐变，中心亮金 `#FFD700`，向外过渡到古铜金 `#B8860B` → `#8B6914`。
- **金牛**：深古铜金 `#8B4513` / `#A0522D` 为主，牛角从根部深古铜渐变到尖端亮金，并加 `#FFEB8C` 高光脊线与 `#FFFAE0` 镜面反光点。
- **文字**：浅金色/近白色 `#FFF8DC`，带轻微投影确保可读性。

### 品牌文字

- 图标底部居中放置 “BullNook” 英文品牌名。
- 使用现代无衬线大写字体风格（代码绘制时通过 Pillow 默认无衬线字体实现）。

## 技术方案

- **实现方式**：Python Pillow 代码绘制。
- **核心绘制逻辑**：
  1. 创建 1024×1024 画布。
  2. 绘制金色径向渐变背景。
  3. 绘制正面牛头多边形，使用径向渐变模拟金属体积感，并添加两侧边缘光。
  4. 绘制向上翻起的牛角：分段贝塞尔曲线 + 渐变底色 + 高光脊线 + 镜面反光点。
  5. 添加眉骨、眼睛、鼻孔、鼻环、嘴线等面部细节。
  6. 添加底部阴影增强立体感。
  7. 绘制 “BullNook” 文字。
  8. 导出为 `app_icon_1024.png`（覆盖原文件），同时保留 Python 源文件便于后续迭代。

## 输出文件

- `app_icon_1024.png`：最终 1024×1024 应用图标。
- `app_icon_1024_horn.png`：旧版抽象牛角图标备份。
- `scripts/generate_app_icon.py`：图标生成脚本，便于后续调整颜色、姿态、文字。
- `tests/test_app_icon.py`：图标生成测试。

## 验收标准

- [x] PNG 尺寸为 1024×1024。
- [x] 图标中清晰可见正面金牛头像，牛角尖端朝上。
- [x] 文字 “BullNook” 在金色背景上清晰可读。
- [x] 整体色调以金色/古铜金为主，无突兀颜色。
- [x] 生成脚本可重复运行并产出一致结果。
- [x] 所有测试通过。
