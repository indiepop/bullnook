# BullNook 红色冲锋牛图标重设计

> 设计日期：2026-07-18
> 状态：待实现

## 目标

将 BullNook（牛角尖）iOS App 的现有图标从“橙黄渐变背景 + 正面卡通牛头”重设计为“红色主体 + 侧面冲锋牛”风格，使牛的形象更卡通、大胆、夸张，并贴合 A 股“红涨牛气”的视觉寓意。

## 设计决策

### 整体构图

- 画布尺寸：1024 × 1024 px（正方形，Xcode 自动处理圆角遮罩）。
- 主体位于画面中心偏右，牛侧身向右上方冲锋，充满动势和力量感。
- 牛占据画面约 60%-70%，确保小尺寸下图标仍可辨识。

### 风格

- **扁平卡通 + 粗描边**：线条简洁、符号感强，适合 App Icon 小尺寸识别。
- **夸张比例**：牛角巨大前弯、肌肉块突出、四肢蹬地、尾巴上扬，强调“冲劲”。
- 视觉关键词：力量、喜庆、卡通、冲击力、牛市。

### 牛姿态

- **侧面冲锋**：牛头低伏、肩部隆起、前腿发力后蹬，整体呈向上冲刺姿态。
- **牛角**：巨大、向前弯曲、金色带高光，强化攻击性和品牌记忆点。
- **表情**：圆睁大眼 + 扬起眉毛 + 喷气鼻孔，卡通化的愤怒/自信感。
- **尾巴**：向上扬起，末端毛发夸张散开，增强动感。

### 色彩

- **牛身主色**：鲜艳正红 `#E60012`，阴影用暗红 `#B3000F`，高光用亮红 `#FF4D4D`。
- **牛角/鼻环**：金色 `#FFD700`，加 `#FFEB8C` 高光与 `#B8860B` 阴影。
- **背景**：深红 `#8B0000` 到暗金 `#B8860B` 的径向渐变，中心亮、四周暗，突出主体。
- **速度线/粒子**：金色 `#FFD700` 短线和星点，营造“涨”的动势。
- **描边/细节**：深褐/黑色 `#3D0C02` 粗描边，增强卡通感和边界清晰度。

### 辅助元素

- 牛身底部和后方添加金色速度线，暗示快速上冲。
- 画面四角点缀少量金色火花/星点，增加喜庆氛围。
- 不添加品牌文字，让牛形象本身成为绝对视觉焦点。

## 技术方案

- **实现方式**：基于现有 Python Pillow 脚本 `generate_icon.py` 重写绘制逻辑。
- **核心绘制步骤**：
  1. 创建 1024×1024 画布，绘制深红到暗金径向渐变背景。
  2. 绘制背景速度线和金色星点。
  3. 用多边形 + 椭圆 + 贝塞尔曲线构建侧面牛身体、头部、四肢、尾巴。
  4. 用渐变填充（多层同形状叠加）模拟红色身体的明暗体积感。
  5. 绘制金色牛角、鼻环，并添加高光。
  6. 绘制眼睛、眉毛、鼻孔喷气等面部表情。
  7. 添加整体描边和底部阴影。
  8. 导出为 `app_icon_cartoon_bull.png`，并同步更新 `BullNook/Assets.xcassets/AppIcon.appiconset/AppIcon.png`。

## 输出文件

- `app_icon_cartoon_bull.png`：最终 1024×1024 应用图标。
- `generate_icon.py`：更新后的图标生成脚本，便于后续颜色/姿态迭代。
- `BullNook/Assets.xcassets/AppIcon.appiconset/AppIcon.png`：Xcode 使用的 App Icon 资源。

## 验收标准

- [ ] PNG 尺寸为 1024×1024。
- [ ] 图标主体为红色侧面冲锋牛，形象卡通、大胆、夸张。
- [ ] 背景以深红/暗金为主，突出红色牛身。
- [ ] 小尺寸（如 60pt）下图标仍清晰可辨。
- [ ] 生成脚本可重复运行并产出一致结果。
- [ ] Xcode 资源文件已同步更新。
