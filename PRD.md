# Bullnook - A股选股App

> 中文名：牛角尖  
> 英文名：Bullnook  
> 平台：iOS (SwiftUI)  
> 目标：每天为用户精选 5 只 A 股，并提供深度分析。推荐逻辑在 iOS 端本地完成。

> 数据策略：允许用户先看到缓存的旧推荐，后台异步刷新生成新推荐。历史推荐会保存并显示从推荐日至今日的涨跌幅。

---

## 一句话定位

**Bullnook 是一款专注于“每日精选 5 只 A 股”的 iOS 选股 App。**  
它区别于传统行情软件的信息轰炸，通过板块热度、龙虎榜资金、个股走势、消息面四维综合分析，为散户投资者提供一份可快速决策的“早盘作战清单”。

---

## 核心产品原则

1. **少即是多**：每天只推荐 5 只股票，降低选择焦虑。
2. **可解释**：每只推荐股必须展示“为什么入选”，包括板块、龙虎榜、走势、消息链四维理由。
3. **可行动**：推荐 → 详情 → 加自选，链路清晰。
4. **不喊单**：只提供分析参考，不承诺收益，强调风险。

---

## 用户画像

- 25-40 岁 A股散户
- 没有足够时间盯盘和深度研究
- 厌倦了信息过载的行情软件
- 希望每天开盘前有一个“今日重点关注清单”

## MVP 功能范围

### P0：每日推荐
- 用户每天首次打开 App 时，后台生成 5 只精选股票（目标 08:00 后）
- 如果有缓存的旧推荐，先展示旧推荐，后台异步刷新
- 展示推荐列表：股票名称、代码、入选理由摘要、从推荐日至今的涨跌幅
- 点击可进入个股详情
- 支持手动下拉刷新，重新生成最新推荐
- 推荐过滤：排除退市股、科创板（688 开头）、北交所（8 开头）

### P0：个股详情
- 股票基本信息（名称、代码、行业、市值）
- F10 关键指标（市盈率、市净率、ROE、营收/净利润增速）
- K线图（日线、周线、月线切换）
- 入选理由拆解（板块、龙虎榜、走势、消息链）
- 加入/移出自选股

### P0：自选股管理
- 自选股列表
- 实时行情刷新（每 5 分钟）
- 涨跌幅、最新价显示
- 点击进入个股详情

### P1：推荐历史
- 保存所有历史推荐记录到本地数据库
- 推荐总表：按日期展示每日 5 只推荐，每只股票显示从推荐日至今的涨跌幅
- 支持按日期筛选、按涨跌幅排序
- 点击单只股票可跳转详情页

### P2：预警通知
- 自选股价异动提醒
- 每日推荐更新提醒

## 数据更新策略

| 数据类型 | 更新频率 | 来源 |
|---------|---------|------|
| 每日推荐 | 用户每天首次打开时生成（目标 08:00 后） | iOS 本地计算 + LLM 生成 |
| 日线/周线/月线 | 每日首次需要时刷新 | 新浪 / 东方财富 |
| 龙虎榜 | 每日收盘后 | 东方财富 |
| 板块热点 | 每日收盘后 | 东方财富 |
| 个股新闻/公告 | 生成推荐前抓取 | 东方财富 / 新浪财经 |
| 自选股行情 | 每 5 分钟 | 新浪 / 腾讯实时接口 |
| F10 财务数据 | 季度/年度报告更新 | 东方财富 |

## LLM API Key 配置

- 未配置时，首次打开 App 弹出配置弹窗
- 也可以在「设置」页中随时配置或修改
- API Key 使用 Keychain 安全存储
- 支持 DeepSeek / Kimi / 通义千问 三种服务商
- 未配置 API Key 时：
  - 仍可查看自选股行情和个股详情
  - 推荐功能显示 "需要配置 LLM API Key 才能生成推荐理由"
  - 推荐列表可展示基于规则的评分结果，但缺少 LLM 分析文本

## 推荐评分逻辑（iOS 本地）

每天用户首次打开 App 时（或手动刷新），iOS 端本地综合以下四个维度评分：

### 1. 板块热度（25%）
- 所属申万/同花顺板块近 5 日涨幅
- 板块资金净流入
- 板块内涨停股数量

### 2. 龙虎榜资金（25%）
- 近 5 日是否上榜
- 机构/游资净买入额
- 知名席位参与情况

### 3. 个股走势（25%）
- 近 5/10/20 日涨跌幅
- 成交量/换手率变化
- 是否突破关键均线（MA5/MA10/MA20）
- MACD / RSI 等简单技术指标

### 4. 消息链（25%）
- 近 7 日个股新闻/公告数量
- 利好消息密度（行业政策、公司公告）
- 与当前市场主线的匹配度

### 5. 推荐过滤规则

推荐候选池需排除以下股票：
- 退市股（股票名称中包含 "退"、"ST" 的股票）
- 科创板：代码以 `688` 开头
- 北交所：代码以 `8` 或 `4` 开头
- 停牌股（无最新成交数据）

可选过滤：
- 上市不满 60 日的新股
- 当日涨跌幅超过 10%（创业板/科创板为 20%）的股票，避免追高

最终按综合得分排序，取前 5 名。LLM 负责对每个入选股票生成一段自然语言“推荐理由”。

## 技术架构概要

- **iOS 端**：SwiftUI + Widget Extension
- **数据与计算**：iOS 本地完成，直接调用新浪/东方财富/腾讯等公开接口
- **本地存储**：SwiftData
- **LLM**：DeepSeek / Kimi / 通义千问（iOS 端直接调用 API）
- **可选后端**：未来仅用于推送通知、版本兼容数据聚合、用户反馈
