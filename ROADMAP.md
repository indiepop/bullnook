# ROADMAP.md

Bullnook 产品迭代路线图。

---

## 阶段 0：MVP（2-3 周）

目标：可运行的最小可用版本，验证核心假设。

后端：
- [ ] 搭建 FastAPI 项目骨架
- [ ] 接入 AkShare，实现日线/周线/月线抓取
- [ ] 实现龙虎榜、板块数据抓取
- [ ] 实现推荐评分引擎（四维度打分）
- [ ] 接入 LLM，生成推荐理由
- [ ] 实现每日推荐定时任务（08:00）
- [ ] 实现自选股行情接口

iOS：
- [ ] 创建 SwiftUI 项目，配置 Widget Extension
- [ ] 实现每日推荐列表页
- [ ] 实现个股详情页（K线图、F10、分析）
- [ ] 实现自选股管理
- [ ] 实现今日推荐小组件
- [ ] 实现网络请求、缓存、错误处理

交付标准：
- 用户每天打开 App 能看到 5 只推荐
- 能点击看详情、K线图、F10
- 能加自选
- 小组件显示 Top 1

## 阶段 1：体验优化（第 4-5 周）

- [ ] 推荐历史页（过去 7 天/30 天）
- [ ] 推荐股后续表现追踪
- [ ] 自选股推送通知（价格异动）
- [ ] 每日推荐更新通知
- [ ] 下拉刷新、加载状态优化
- [ ] 暗色/浅色模式切换
- [ ] 分享功能：分享推荐股票卡片

---

## 阶段 2：策略增强（第 6-8 周）

- [ ] 增加更多评分因子：
  - 北向资金流入
  - 融资融券余额变化
  - 市场情绪指标
- [ ] 推荐结果可解释性增强（四维度雷达图）
- [ ] 用户可自定义筛选偏好（如只看消费/科技板块）
- [ ] 后端推荐策略 A/B 测试

---

## 阶段 3：商业化探索（第 9-12 周）

- [ ] 用户账户系统
- [ ] 推荐历史无限制查看
- [ ] 高级版：更多推荐数量、实时预警、深度研报
- [ ] 付费订阅（Apple In-App Purchase）
- [ ] 邀请/推荐奖励机制

---

## 长期方向

- 接入更多资产类别（港股、ETF、可转债）
- 投资组合跟踪与收益分析
- 社交化：用户可分享观点但不构成投资建议
- 本地化海外市场（美股中概股）
